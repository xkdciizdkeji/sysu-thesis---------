\begin{abstract}
  随着先进工艺节点的持续演进，超大规模集成电路（VLSI）设计规模与复杂度呈指数级增长，传统电子设计自动化（EDA）工具在计算效率与设计质量方面面临严峻挑战。布局与布线作为后端设计流程中的核心环节，其结果直接影响芯片的性能、功耗与面积（PPA）。全局布线问题具有强组合优化特性与复杂约束条件，属于典型的 NP 难问题；同时，传统以串行处理为主的算法难以充分利用现代并行计算平台的算力优势，在处理千万级以上规模电路时面临速度与质量的双重瓶颈。在布局环节，由于设计空间极大，为了提升速度，解析式布局算法需要对大量约束进行一定程度的简化，由此产生了与真实布局中难以处理的细粒度约束之间错配。

  针对上述问题，本文围绕基于并行计算及模块级布局规划优化的 VLSI 高效布局布线算法的课题，从并行布线加速的初始布线工作流和模块级布局规划优化工作流两个子课题展开研究，实现面向超大规模电路设计的高效全局布线算法和布局优化算法。首先，在初始布线阶段，提出一套应用于全局布线阶段的动态拥塞感知初始布线工作流。该方法在生成树构建与路径选择过程中引入实时更新的拥塞代价模型，使布线决策能够动态反映当前全局资源分布状态，从而缓解传统串行布线算法中静态代价函数导致的局部最优问题。针对超大规模电路中全局布线的计算效率问题，本文使用了一种面向并行计算平台的全局布线加速方法。该方法从数据级并行与线网级并行两个层面出发，对布线流程进行重构：一方面，使用GPU加速的矩阵运算流程实现了对全局布线资源的高效并行计算；另一方面，同时考虑所有线网的综合布线代价，实现了全局视角的综合衡量，在保证布线质量的前提下显著提升算法运行效率。
  此外在布局阶段，为了缓解粗粒度解析布局算法精确度不足、细粒度全局搜索方法解空间极大的问题，本文研究了基于多段模拟退火算法的细粒度布局优化算法，通过合理设置领域选择以及多段优化策略，可提升布局设计空间探索的效率，从而高效实现对模块空间分布的细粒度动态调节。

本文研究了拥塞感知的初始布线工作流以及基于多段模拟退火的模块级布局规划细粒度优化算法，分别应用并行计算以及多段模拟退火，在VLSI布局布线设计场景下权衡了质量与计算效率。实验测试表明，应用了拥塞感知初始布线工作流的全局布线算法在公开基准上显著改善拥塞指标，在线长、通孔指标不退化的情况下，拥塞指标相比DGR与CUGR2分别取得约24.9\%和44.1\%的优化；提出的模块级布局优化算法在公开基准上取得了约3.86\%的线长优化。

\end{abstract}

\begin{abstract*}
  With the continuous evolution of advanced process nodes, the scale and complexity of very-large-scale integration (VLSI) designs have grown exponentially, posing severe challenges to traditional electronic design automation (EDA) tools in both computational efficiency and design quality. As core stages in backend design, placement and routing directly affect power, performance, and area (PPA). Global routing exhibits strong combinational optimization characteristics and complex constraints, and is a typical NP-hard problem. Meanwhile, conventional serial-dominated algorithms cannot fully exploit the computational power of modern parallel computing platforms, resulting in dual bottlenecks in runtime and solution quality when handling circuits at the ten-million-instance scale and beyond. In placement, because the design space is extremely large, analytical placement algorithms often simplify a large number of constraints to improve speed, which leads to a mismatch with fine-grained constraints that are difficult to handle in real placement.

  To address these issues, this thesis investigates efficient VLSI placement-and-routing algorithms based on parallel computing and module-level placement-planning optimization. The study is organized into two subtopics: an initial routing workflow with parallel-routing acceleration and a module-level placement-planning optimization workflow, enabling efficient collaborative optimization for ultra-large-scale circuit designs. First, in the initial routing stage, a dynamic congestion-aware initial routing workflow for global routing is proposed. The method introduces a real-time updated congestion cost model into Steiner tree construction and path selection, allowing routing decisions to dynamically reflect the current global resource distribution and thereby alleviating local-optimum issues caused by static cost functions in conventional serial routing algorithms. To improve the computational efficiency of global routing in ultra-large-scale circuits, this thesis further introduces a global-routing acceleration method oriented to parallel computing platforms. From both data-level and net-level parallelism, the routing flow is reconstructed: on the one hand, a GPU-accelerated matrix-computation pipeline enables efficient parallel computation of global-routing resources; on the other hand, by jointly considering the comprehensive routing cost of all nets, the method performs global-view evaluation and significantly improves runtime while maintaining routing quality.

  In addition, in the placement stage, to mitigate the limited precision of coarse-grained analytical placement algorithms and the huge search space of fine-grained global search methods, this thesis studies a fine-grained placement optimization algorithm based on multi-stage simulated annealing. By properly designing neighborhood selection and multi-stage optimization strategies, the method improves the efficiency of placement design-space exploration and enables fine-grained dynamic adjustment of module spatial distribution.

  This thesis studies a congestion-aware initial routing workflow and a fine-grained module-level placement-planning optimization algorithm based on multi-stage simulated annealing. By respectively leveraging parallel computing and multi-stage simulated annealing, the proposed methods balance solution quality and computational efficiency in VLSI placement-and-routing scenarios. Experimental results show that, on public benchmarks, the global-routing algorithm with the congestion-aware initial routing workflow significantly improves congestion. Without degrading wirelength and via metrics, congestion is improved by about 24.9\% and 44.1\% compared with DGR and CUGR2, respectively. The proposed module-level placement optimization algorithm achieves about a 3.86\% wirelength improvement on public benchmarks.
\end{abstract*}
